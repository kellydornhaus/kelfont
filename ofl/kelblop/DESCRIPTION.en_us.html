Kelblop is a bold, geometric display typeface with rounded forms and a friendly, approachable character. Designed as a companion to Kelblok, it shares the same heavy weight and compact proportions while offering a softer, more organic feel. Kelblop is ideal for headlines, game interfaces, and playful branding. It supports the full Latin character set for broad multilingual coverage.

To contribute, see github.com/kellydornhaus/kelfont.
